External CSS

External CSS helps you make the webpage look exactly how you want. This can help you develop a unique and personalized webpage. The external css makes rules that the webpage has to follow. This allows the page to look a certain way. The rule is applied to everysingle one of the elements in the webpage. This also redeuces the way of creating errors.
